## CLAUDE CODE INSTRUCTIONS

When building any page:

### Step 1: Fetch & Analyze
```
For each competitor URL listed for this page:
1. Fetch the page content
2. Note the section structure (H1, H2s, content blocks)
3. Identify CTAs and their placement
4. Note any unique elements (calculators, videos, animations)
```

### Step 2: Extract Patterns
```
Across all competitors, identify:
- Common sections that appear on most pages
- Section ordering patterns
- Content length and depth
- Visual elements (screenshots, icons, videos)
- Social proof placement
```

### Step 3: Adapt for Brahmin
```
Build the page with:
- Brahmin's voice (practical, founder-led, no-nonsense)
- Brahmin's differentiators woven in naturally
- Appropriate length for $1M-$50M target market
- Clear CTAs (Book a Demo as primary)
```

### Step 4: Quality Check
```
Verify:
- All sections have clear purpose
- No fluff or filler content
- Mobile-responsive design
- Consistent with design system (see INSTRUCTIONS.md)
```
